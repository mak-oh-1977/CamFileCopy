=============================================================
カメラファイルコピー ver.0.3
=============================================================

機能
デジタルカメラのメモリーカードやポータブルストレージに存在する、
画像ファイルをまとめてパソコンに取り込みます。
取り込んだ画像は指定したフォルダにひとまとめで入れる他、
日付ごとのフォルダに振り分けることもできます。

特徴 
シンプル操作。 
忙しい現場でも、設定済みであれば１クリックでコピー完了。 
JPEGファイルの他、Canon EOSシリーズRAWファイル(CRW,CR2)にも対応。 
サブフォルダにある画像もまとめて取得。 
Windowsにドライブとして表示されるタイプのデバイスであれば動作可能。 
フリーソフトです。 

動作環境 
Windows XP/2000

インストール
解凍したファイルをダブルクリックすれば起動します（インストーラ等はありません） 

アンインストール
インストール時に出来たファイルと
設定ファイル(CamFileCopy.ini)を削除すればいいです（レジストリ等は使用していません）。 

使い方 
１．コピー元フォルダ（メモリーカードを入れたドライブ等）を指定してください。 
２．コピー先フォルダを指定してください。 
３．コピー先のフォルダ名称タイプを指定してください（固定名称か日付）。 
４．フォルダ名称タイプを固定名称にした場合は名称を、日付の場合は書式を指定してください。 
５．開始ボタンを押してください。 
６．コピー中のファイルが表示されるのでしばしお待ちください。 
７．中断ボタンを押すことで途中で処理を終了することも出来ます
　　（ファイルのコピー中の場合は、中断を押した時にコピーしているファイルまでコピーされます）。
８．全てコピーされれば終了です。 
９．設定は保存されるので、設定を変えなくてよい場合は、次回からは１．〜４．は省略できます。 

その他 
日付ごとのフォルダ振り分けはファイルの更新日付を見ています。Exif情報は見ていませんので、
撮影した日付と異なる場合があります。 
コピー先フォルダに同名のファイルが既にある場合は、コピーを行いません。
